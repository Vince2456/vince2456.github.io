EmailJS Template Setup Guide
==========================

1. Template Name: Portfolio Contact Form
2. To Email: kquophilyrical@gmail.com
3. Subject: {{subject}}

Email Content (copy this exactly):

Hello,

You have received a new message from your portfolio contact form:

From: {{from_name}} ({{from_email}})
Subject: {{subject}}

Message:
{{message}}

---
Sent from: Amoakohene Vincent's Portfolio Website
Date: {{sent_at}}

4. IMPORTANT: Make sure these variables are available:
   - {{from_name}}
   - {{from_email}}
   - {{subject}}
   - {{message}}
   - {{sent_at}} (optional, for timestamp)

5. After creating the template:
   - Copy the Template ID (looks like: template_xxxxxxx)
   - Update your script.js with the correct Template ID

6. Test the template in EmailJS dashboard before using it on your site.
